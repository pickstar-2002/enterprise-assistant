# 企业级 AI 数字员工 - HR/IT 智能助手

基于魔珐星云3D数字人技术，打造一个面向企业场景的智能数字员工，专注于 HR 政策解答和 IT 故障报修两大核心场景。

## 功能特色

- **密钥管理**：首次访问引导配置密钥，支持内置测试密钥快速体验
- **连接控制**：手动控制数字人连接/断开，避免积分浪费
- **自定义知识库**：用户上传文档，自动提取文本、分块、向量化存储
- **HR 政策解答**：薪资福利、考勤休假、招聘入职、培训发展等政策咨询
- **IT 故障报修**：硬件故障、软件问题、网络问题、账号权限等技术支持
- **多模态对话**：支持图片上传、截图粘贴、图文混合对话
- **工单创建**：自动生成故障工单，支持工单状态跟踪

## 技术栈

- 前端：React 18 + TypeScript + Vite + Zustand + TailwindCSS
- 后端：Express + TypeScript
- AI：魔搭社区 Qwen3-VL（多模态对话）+ Qwen3-Embedding-8B（向量化）
- 数字人：魔珐星云具身驱动SDK

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

## 环境变量

参见 `.env.server` 配置魔搭 AI 密钥和数字人密钥。
